# Цели

- Перенаправление потоков (`>`, `>>`, `2>`, `&>`)
- Конвейеры (`|`)
- Поиск файлов (`find`, `grep`)
- Управление процессами (фон, `jobs`, `kill`)
- Анализ диска (`df`, `du`)

# Перенаправление

- `>` : перенаправить stdout (перезапись)
- `>>` : перенаправить stdout (добавление)
- `2>` : перенаправить stderr
- `&>` : перенаправить stdout и stderr

# Конвейеры (pipe)

- `команда1 | команда2`
- Пример: `ls -l | grep ".conf"`

# Поиск файлов

- `find` : поиск по имени, типу, размеру
- `grep` : поиск по содержимому
- `grep -r` : рекурсивный поиск

# Процессы

- `ps aux` : список всех процессов
- `&` : запуск в фоне
- `jobs` : список задач оболочки
- `kill PID` : завершить процесс

# Анализ диска

- `df -h` : свободное место на дисках
- `du -sh ~` : размер домашнего каталога
