# 1. Цель работы

Изучить основы программирования в оболочке bash. Научиться писать командные файлы, использовать переменные, арифметические операции, управляющие конструкции (for, case, if, while), обрабатывать аргументы командной строки.

# 2. Задание

1. Написать скрипт, который делает резервную копию самого себя в каталог `~/backup` с архивацией.
2. Написать скрипт, обрабатывающий произвольное число аргументов командной строки (более 10).
3. Написать скрипт — аналог команды `ls` (без использования `ls` и `dir`), выводящий права доступа и имена файлов.
4. Написать скрипт, подсчитывающий количество файлов с заданным расширением в указанном каталоге.

# 3. Выполнение лабораторной работы

## 3.1. Скрипт резервного копирования

**Листинг `backup_self.sh`** :

```bash
#!/bin/bash
BACKUP_DIR=~/backup
mkdir -p "$BACKUP_DIR"
cp "$0" "$BACKUP_DIR/$(basename "$0").bak"
tar -czf "$BACKUP_DIR/backup_self.tar.gz" "$0"
echo "Sauvegarde effectuée dans $BACKUP_DIR"
