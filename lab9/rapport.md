# 5. Ответы на контрольные вопросы

**1. Какие режимы работы есть в mc? Охарактеризуйте их.**  
mc имеет следующие режимы отображения панелей: Standard (стандартный список файлов), Info (информация о файле и системе), Tree (дерево каталогов), Quick View (быстрый просмотр файла).

**2. Какие операции с файлами можно выполнить как с помощью команд shell, так и с помощью меню mc? Приведите примеры.**  
Копирование (cp → F5), перемещение (mv → F6), удаление (rm → F8), изменение прав доступа (chmod → Ctrl+x c), создание ссылок (ln → Ctrl+x s).

**3. Опишите структуру меню левой (или правой) панели mc, дайте характеристику командам.**  
Меню панели содержит: List mode (режим списка), Info mode (режим информации), Tree mode (режим дерева), Quick View (быстрый просмотр), Filter (фильтр), Re-read (перечитать), Sort order (сортировка).

**4. Опишите структуру меню File mc, дайте характеристику командам.**  
Меню File включает: New file (новый файл), Copy (копирование), Move (перемещение), Delete (удаление), Rename (переименование), Link (ссылка), Edit (редактирование), View (просмотр), Chmod (права доступа), Chown (владелец).

**5. Опишите структуру меню Command mc, дайте характеристику командам.**  
Меню Command содержит: Find file (поиск файлов), Directory tree (дерево каталогов), Swap panels (переставить панели), Compare directories (сравнить каталоги), Command line history (история команд), Directory hotlist (каталоги быстрого доступа).

**6. Опишите структуру меню Options mc, дайте характеристику командам.**  
Меню Options включает: Configuration (конфигурация), Appearance (внешний вид), Panel options (настройки панелей), Confirmation (подтверждения), Learn keys (обучение клавишам), Virtual FS (виртуальная ФС).

**7. Назовите и дайте характеристику встроенным командам mc.**  
Встроенные команды: F1 (Help), F2 (User menu), F3 (View), F4 (Edit), F5 (Copy), F6 (Move/Rename), F7 (Mkdir), F8 (Delete), F9 (Menu), F10 (Quit), Ctrl+o (Shell output), Ctrl+u (Swap panels).

**8. Назовите и дайте характеристику командам встроенного редактора mc.**  
Встроенный редактор поддерживает: Ctrl+y (удалить строку), Ctrl+k (вырезать до конца строки), Ctrl+u (отменить), Ctrl+Insert (копировать), Shift+Insert (вставить), F2 (сохранить), F4 (сохранить и выйти).

**9. Дайте характеристику средствам mc, которые позволяют создавать меню, определяемое пользователем.**  
Файл `~/.config/mc/menu` (или `~/.mc/menu`) позволяет создавать пользовательские меню, вызываемые клавишей F2.

**10. Дайте характеристику средствам mc, которые позволяют выполнять действия, определяемое пользователем, над текущим файлом.**  
Расширения файлов задаются в файле `~/.config/mc/ext.d/`. Можно определить, какая программа запускается для файлов с определённым расширением.
